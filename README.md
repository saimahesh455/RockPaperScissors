# 🎮 Rock Paper Scissors Game

A modern Rock Paper Scissors game built with **Python** and **Streamlit**.

## 🚀 Live Demo

👉 https://rockpapersciapprs-ati879augrz5sz42scpee2.streamlit.app/

## ✨ Features

- 🪨 Rock, 📄 Paper, ✂️ Scissors PNG images
- 🎵 Background music
- 🔊 Win, Lose, and Draw sound effects
- ⏳ Animated countdown
- 🏆 Best of 3 / Best of 5 / Best of 7 modes
- 📊 Live scoreboard
- 📜 Match history
- 📈 Win statistics
- 🔄 Reset Game
- 🌙 Modern dark UI

## 🛠️ Installation

```bash
git clone https://github.com/saimahesh455/RockPaperScissors.git
cd RockPaperScissors
pip install -r requirements.txt
streamlit run app.py
```
